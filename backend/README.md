# Recipe Recommendation System

Step 1: Install Docker Desktop

Step 2: Run docker desktop

Step 3: Open terminal -> git clone https://github.com/buinguyenkhai/pttkht-20251-recipe-recommendation-system.git

Step 4: cd pttkht-20251-recipe-recommendation-system

Step 5: git checkout duc/database

Step 6: docker-compose build

Step 7: docker-compose up -d

Open: localhost:8000/recipes
